Gestor de Pedidos

Permite agregar, editar y eliminar pedidos, generar reportes de ventas y controlar el inventario en tiempo real. Utiliza una interfaz sencilla y fácil de usar con Tkinter (Python), y SQlite para la base de datos.

GitHub: https://github.com/brancorc/GestionCookies

Dashboard de Ventas en Excel
